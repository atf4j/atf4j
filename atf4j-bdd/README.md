# atf4j-bdd
## Automation Test Framework for Java

## Maven Dependency

		<dependency>
			<groupId>net.atf4j</groupId>
			<artifactId>atf4j</artifactId>
			<version>0.3.0-SNAPSHOT</version>
		</dependency>

## Usage

In command line shell run:

    mvn test

## Overriding options

    mvn -Doptions="value" test
