# atf4j-webdriver

## Automated Test Framework for Java Selenium WebDriver

## Maven Dependency

	<dependency>
		<groupId>net.atf4j</groupId>
		<artifactId>atf4j-webdriver</artifactId>
		<version>0.4.0-SNAPSHOT</version>
	</dependency>

## Usage

In command line shell run:

    mvn test

## Overriding options

    mvn -Doptions="value" test
